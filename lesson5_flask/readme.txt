-------------------------------------
Prerequisites
- Python - 3.9
- virtualenv
- Pip
Installing

Make a new virtualenv (you may as well not if you know what you're doing)
$ virtualenv -p python3 venv

Clone this repo
$ git clone (url for this repo)

Install requirements
$ pip install -r requirements.txt

Using
Run
$ flask run
